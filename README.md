# DAO Voting and Governance (Web3)

This project simulates a simple DAO voting system.

It helps beginners understand how proposals and voting work in decentralized organizations.

## What this project does

- Creates proposals
- Stores votes
- Counts results
- Shows winning option

## Why this project is useful

You will learn:
- How DAOs work
- How on-chain voting is structured
- How governance logic is written
- How to organize a blockchain project

## Tech used

- JavaScript
- Node.js
- DAO concepts
- Blockchain logic (simulated)

## How to run

1. Install Node.js  
2. Clone this repo  
3. Open terminal in project folder  
4. Run: `node app.js`  

## Files overview

- `app.js` → Main app file  
- `proposal.js` → Proposal logic  
- `vote.js` → Voting logic  
- `result.js` → Vote counting  
- `member.js` → Member validation  
- `storage.js` → Data storage  
- `config.js` → Settings  
- `logger.js` → Logs  
- `sample.env` → Config example  
- `package.json` → Project info
